# Helper to set streaming to TRUE on large data sets

Helper to set streaming to TRUE on large data sets

## Usage

``` r
auto_streaming(n_cells, streaming = NULL, .verbose = TRUE)
```

## Arguments

- n_cells:

  Integer. Number of cells in the data set.

- streaming:

  Optional boolean. Parsed from the main function.

- .verbose:

  Boolean. Controls verbosity of the function.

## Value

Boolean if streaming should be used.
