# Check cell marker list

Checkmate extension for checking a cell marker list as returned by
[`prepare_cell_markers()`](https://gregorlueg.github.io/bixverse/reference/prepare_cell_markers.md).

## Usage

``` r
checkCellMarkerList(x)
```

## Arguments

- x:

  The list to check.

## Value

`TRUE` if the check was successful, otherwise an error message.
