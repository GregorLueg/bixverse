# Extract and concatenate a named field from a list of group results

Extract and concatenate a named field from a list of group results

## Usage

``` r
.concat_per_cell(group_results, field)
```

## Arguments

- group_results:

  Named list of per-group result objects.

- field:

  Character. Name of the field to extract from each element.

## Value

Unnamed vector of concatenated values.
