# Read named obs columns in cells_to_keep order, return a data.table.

Read named obs columns in cells_to_keep order, return a data.table.

## Usage

``` r
.read_label_columns(sc_object, columns)
```

## Arguments

- sc_object:

  `SingleCells` class

- columns:

  String. The columns to extract

## Value

A data.table with the desired columns.
