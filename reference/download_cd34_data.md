# Download the CD34 example data from SEACells

This function downloads the CD34 data set from the SEACells paper into
the temperorary directory.

## Usage

``` r
download_cd34_data(quiet = FALSE)
```

## Arguments

- quiet:

  Boolean. If the download shall be quiet.

## Value

String. The path to the extracted doublet detection data.

## References

Persad, et al., Nat. Biotechnol., 2023
