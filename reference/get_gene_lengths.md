# Get the gene lengths

Get the gene lengths

## Usage

``` r
get_gene_lengths(x, species = c("human", "mouse", "rat"), ...)
```

## Arguments

- x:

  Object to extract gene lengths from. Can be a matrix or bulk_coexp
  object.

- species:

  String. One of `c("human", "mouse", "rat")`.

- ...:

  Additional parameters passed to methods.
