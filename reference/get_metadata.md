# Return the metadata

Getter function to extract the metadata from the
[`BulkCoExp()`](BulkCoExp.md) or [`BulkDge()`](BulkDge.md).

## Usage

``` r
get_metadata(object, ...)
```

## Arguments

- object:

  The underlying object, either [`BulkCoExp()`](BulkCoExp.md) or
  [`BulkDge()`](BulkDge.md).

- ...:

  Additional arguments to parse to the functions.

## Value

Returns the metadata stored in the class.
