# Get the RBH results

Pulls out the RBH results if you ran
[`generate_rbh_graph()`](generate_rbh_graph.md)

## Usage

``` r
get_rbh_res(object)
```

## Arguments

- object:

  The underlying class [`rbh_graph()`](rbh_graph.md).

## Value

The data.table with the RBH result if found, otherwise NULL.
