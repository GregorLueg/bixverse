# Run Harmony

A version of Harmony by Korsunsky et al., implemented in Rust.

## Usage

``` r
harmony_sc(
  object,
  batch_column,
  additional_batch_columns = NULL,
  harmony_params = params_sc_harmony(),
  seed = 42L,
  .verbose = TRUE
)
```
