# Calculates the compactness of the MetaCells based on diffusion map coordinates

**\[experimental\]** Calculates the meta cell compactness based on the
diffusion map coordinates.

## Usage

``` r
rs_metacell_compactness(dc, meta_cells)
```

## Arguments

- dc:

  Numerical matrix. The diffusion map coordinates.

- meta_cells:

  List. The cell indices of the meta cells.

## Value

The compactness results
