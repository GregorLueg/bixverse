# Calculate the hypergeometric test in Rust

**\[experimental\]**

## Usage

``` r
rs_phyper(q, m, n, k)
```

## Arguments

- q:

  Number of white balls drawn out of urn.

- m:

  Number of white balls in the urn.

- n:

  Number of black balls in the urn.

- k:

  The number of balls drawn out of the urn.

## Value

P-value (with lower.tail set to False)
