# Set the HVG genes

Set the HVG genes

Set the highly variable genes for a `single_cell_exp` class.

## Usage

``` r
set_hvg(x, hvg)

# S3 method for class 'sc_mapper'
set_hvg(x, hvg)
```

## Arguments

- x:

  An object to set the HVGs for

- hvg:

  String or integer. The names or indices of the highly variable genes.
