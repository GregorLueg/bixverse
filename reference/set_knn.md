# Set/add KNN

Set/add KNN

## Usage

``` r
set_knn(x, knn)

## S7 method for class <bixverse::MetaCells>
set_knn(x, knn)

# S3 method for class 'ScCache'
```

## Arguments

- x:

  An object to add the KNN data to

  `SingleCellNearestNeighbour` class to add to the classes.
