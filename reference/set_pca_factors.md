# Set/add PCA factors

Set/add PCA factors

Set the PCA factors for a `single_cell_exp` class.

## Usage

``` r
set_pca_factors(x, pca_factor)

# S3 method for class 'sc_cache'
set_pca_factors(x, pca_factor)
```

## Arguments

- x:

  An object to add the PCA factors for.

- pca_factor:

  Numerical matrix. The matrix with the PCA factors.
