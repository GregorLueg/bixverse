# Add a new column to the var table

Add a new column to the var table

## Usage

``` r
set_sc_new_var_cols(object, data_list)
```

## Arguments

- object:

  [`bixverse::single_cell_exp`](single_cell_exp.md) class.

- data_list:

  Named list with the data to add.

## Value

The class with updated var table in the DuckDB
