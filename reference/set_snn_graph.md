# Set/add KNN

Set/add KNN

Set the sNN graph for a `single_cell_exp` class.

## Usage

``` r
set_snn_graph(x, snn_graph)

# S3 method for class 'sc_cache'
set_snn_graph(x, snn_graph)
```

## Arguments

- x:

  An object to add the KNN data to

- snn_graph:

  Igraph. The sNN graph for subsequent clustering.
